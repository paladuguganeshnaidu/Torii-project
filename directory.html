Torii-Project Directory Structure

Torii-Project/ ├─ .git/ # Git metadata ├─ .gitignore # Ignore venv, DB, caches ├─ .nojekyll # Disable Jekyll on GitHub Pages ├─ .venv/ # Local virtual env (created by start scripts) ├─ .vscode/ │ ├─ html.code-snippets # torii-html / torii-css snippets │ ├─ launch.json # Debug config (debugpy) │ └─ tasks.json # Run task: “Run Torii (PowerShell)” ├─ assets/ │ └─ css/ │ └─ main.css # Root CSS for GitHub Pages + homepage UI ├─ Backend/ │ ├─ app.py # Flask app: factory, routes, API │ ├─ auth.py # Auth blueprint (register/login/logout skeleton) │ ├─ config.py # Paths, SECRET_KEY, DB config │ ├─ database.py # SQLite helpers (init_db/get_db/teardown) │ ├─ requirements.txt # Flask + requests │ ├─ templates/ │ │ ├─ index.html # Server-rendered dashboard (Jinja) │ │ ├─ login.html # Login page │ │ └─ register.html # Register page │ └─ tools/ │ ├─ __init__.py # Re-exports tool functions │ ├─ email_analyzer.py # analyze_email_tool(request) │ ├─ url_scanner.py # scan_url_tool(request) │ ├─ password_cracker.py # crack_hash_tool(request) │ ├─ sms_spam_detector.py # test_sms_tool(request) │ ├─ malware_analyzer.py # analyze_file_tool(request) │ └─ web_recon.py # recon_target_tool(request) ├─ css/ │ ├─ main.css # Original CSS (not used by new homepage) │ └─ responsive.css ├─ database/ │ ├─ app.db # Created at runtime │ └─ schema.sql # Tool logs table schema ├─ index.html # Root homepage (grid tiles + search) ├─ README.md # Overview + run instructions ├─ run.py # Starts Flask dev server ├─ start.bat # Windows cmd start script ├─ start.ps1 # PowerShell start script ├─ tool1-email-analyzer.html # Static single-page demos (legacy) ├─ tool2-url-scanner.html ├─ tool3-password-cracker.html ├─ tool4-sms-spam-tester.html ├─ tool5-malware-analyzer.html └─ tool6-web-recon.html
Legend:
Folders are shown with /.
# Comments describe each file or folder.
├─ and └─ indicate tree branches.
